SymSpellCppPy Docs
==================

A Fast SymSpell v6.5 port for python written in C++ using pybind11.

Detailed documentation about Symspell and its usage and functionalities: `<https://github.com/wolfgarbe/SymSpell>`_

This library essential is a re-rewrite of SymSpell in C++ and ported to python using pybind11, it's designed to match the functionalities of
*symspellpy* another SymSpell port in pure python, this version is in general 3x-40x faster than symspellpy, while providing same functionalities.

Benchmarks are provided in the library homepage on github `<https://github.com/viig99/SymSpellCppPy>`_
